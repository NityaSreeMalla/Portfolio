PROJECT 01

AI Photo Gallery Cleaner

Intelligent duplicate detection & quality filtering pipeline

⭐ Featured Computer Vision
Problem Statement
Photo libraries accumulate thousands of near-identical and blurry images. Manual cleaning is impractical — the challenge was automating intelligent selection of the single best image from duplicate clusters.
Approach
Built a multi-stage pipeline: perceptual hashing for duplicate detection → Laplacian variance for blur scoring → ResNet50 embeddings for visual similarity grouping → best-image selector per cluster.
Key Results
Accurately groups visually similar images and ranks by quality score, retaining only the sharpest, highest-quality copy while removing redundant duplicates.
Key Learnings
Perceptual hashing is orders of magnitude faster than pixel-level comparison; transfer learning with ResNet50 dramatically improves semantic similarity clustering over color histograms alone.
Python OpenCV PyTorch ResNet50 ImageHash NumPy Pillow

PROJECT 02

Bank Loan Approval Prediction

Binary classification model for financial risk assessment

Machine Learning
Problem Statement
Banks process thousands of loan applications — manual review is slow and inconsistent. Built a model to predict loan approval likelihood, reducing decision time while maintaining fairness and accuracy.
Approach
End-to-end pipeline: EDA → missing value imputation → feature engineering (debt-to-income ratios, credit tiers) → model training with cross-validation → evaluation with precision/recall/AUC.
Key Results
Achieved strong predictive performance with classification metrics. Model identifies high-risk applications with meaningful precision, suitable for production-grade decision support.
Key Learnings
Handling class imbalance via SMOTE; importance of feature engineering in financial datasets; precision vs recall tradeoffs in risk-sensitive binary classification.
